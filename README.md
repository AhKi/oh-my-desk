## Getting Started 

This project use [yarn](https://yarnpkg.com/lang/en/) for package manager.

**Project Clone**
```
git clone https://github.com/AhKi/oh-my-desk.git
cd oh-my-desk
```

**Start Project**

```
yarn && yarn start 
```

**Show Unit test coverage**

```
yarn && yarn test
```

**Build**

```
yarn && yarn package
```
